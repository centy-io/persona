# Initialize TypeScript pnpm package structure

## Overview

Set up the foundational npm package structure for the Persona CLI.

## Tasks

-   Initialize `package.json` with proper metadata:
    

\- `name`: `@centy/persona` - `version`: `0.1.0` - `bin` entry pointing to the compiled CLI entry point - `main` / `exports` fields for the compiled output - `files` to include only `dist/` - `engines` field specifying minimum Node.js version

-   Add TypeScript with `tsconfig.json`
    
-   Set up `tsup` (or `esbuild`) as the build tool to compile to CommonJS and/or ESM
    
-   Add `.gitignore`, `.npmignore`
    
-   Add `build`, `dev`, and `prepublishOnly` scripts in `package.json`
    
-   Set `publishConfig.access` to `public` for scoped package
    

## Acceptance Criteria

-   `npm run build` compiles TypeScript successfully to `dist/`
    
-   Package structure is clean and ready for publishing
